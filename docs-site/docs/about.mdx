**Convert-Pheno** is developed at [CNAG](https://cnag.eu), Barcelona, Spain, with contributions from collaborators who helped shape the supported formats, interface design, and documentation.

<div className="about-hero">
  <p>
    The project brings together clinical data modeling, standard format conversion,
    Beacon v2 interoperability, and practical tooling for users working with
    phenotypic data.
  </p>
</div>
